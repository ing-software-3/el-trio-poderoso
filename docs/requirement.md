## 
✅ Requerimientos Funcionales



Registrar productos en el inventario (nombre, categoría, cantidad, precio)
Buscar productos por nombre o categoría
Actualizar información de un producto
Eliminar productos del inventario
Registrar entradas de productos (cuando llegan nuevos)
Registrar salidas de productos (cuando se usan en el jardín)
Mostrar la cantidad disponible de cada producto
Generar alertas básicas cuando el stock esté bajo


## ⚙️ Requerimientos No Funcionales


### Usabilidad:
El sistema debe ser fácil de usar para personal del jardín (no técnicos)
### Rendimiento:
Las consultas deben responder en menos de 2 segundos
### Disponibilidad:
El sistema debe estar disponible cuando el usuario lo necesite
### Seguridad:
Solo usuarios autorizados pueden modificar el inventario
### Escalabilidad:
El sistema debe permitir agregar más productos sin afectar el rendimiento
### Mantenibilidad:
El código debe estar organizado y documentado (uso de módulos en Python)
### Compatibilidad:
Debe funcionar en cualquier navegador mediante la API (FastAPI)
