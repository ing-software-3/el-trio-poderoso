# Requerimientos

## ✅ Requerimientos Funcionales

RF01 -Registrar productos en el inventario (nombre, categoría,
cantidad, precio)
RF02 -Buscar productos por nombre o categoría
RF03 -Actualizar información de un producto
RF04 -Eliminar productos del inventario
RF05 -Registrar entradas de productos (cuando llegan nuevos)
RF06 -Registrar salidas de productos (cuando se usan en el jardín)
RF07 -Mostrar la cantidad disponible de cada producto
RF08 -Generar alertas básicas cuando el stock esté bajo

## ⚙️ Requerimientos No Funcionales

### Usabilidad

El sistema debe ser fácil de usar para personal del jardín (no técnicos)

### Rendimiento

Las consultas deben responder en menos de 2 segundos

### Disponibilidad

El sistema debe estar disponible cuando el usuario lo necesite

### Seguridad

Solo usuarios autorizados pueden modificar el inventario

### Escalabilidad

El sistema debe permitir agregar más productos sin afectar el rendimiento

### Mantenibilidad

El código debe estar organizado y documentado (uso de módulos en Python)

### Compatibilidad

Debe funcionar en cualquier navegador mediante la API (FastAPI)
